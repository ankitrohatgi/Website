Version 3.7 is out!


This version contains a number of minor fixes - especially for Polar Diagrams. Also, now the exported JSON file also contains the actual digitized data instead of just the pixel locations of the points. To read a summary of changes, check out the release notes.


I have also been working on bar charts, grid removal and a Chrome Extension to allow opening any image on any webpage by simply right-clicking on it and selecting "Send to WebPlotDigitizer". Unfortunately, these features were not fully stable at the time of the release, so they have been disabled in this release.
