Multiple Languages

Support for multiple languages has now been added. For now only French and German translations are available. If you wish to translation WebPlotDigitizer to other languages, then feel free to send me an email.


Export All Data

You can now export data from all datasets to a combined CSV file in one single operation. All this requires is clicking the 'Export Data' option in the 'Data' menu.



Extraction Algorithm Improvements

X step with interpolation and X step algorithms have been extended to work with log-scale XY plots.
